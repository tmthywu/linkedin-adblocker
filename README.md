# linkedin-adblocker: a chromium extension that blocks linkedin ads using javascript injection

How to use the extension:

1. Click your browser’s “burger menu” on the top right corner or visit Settings directly
2. Navigate to ‘Extensions’
3. Toggle on “Developer Mode”, which is usually on the top right corner of the ‘Extensions Page’
4. Some buttons should have appeared on the top of the page. Click “Load Unpacked”
5. Select your project directory from the File Browser
6. You’re all set!

If you’ve followed all the steps correctly, you should now have a new icon where you extensions are located, also on the top right.
